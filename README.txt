This project uses toolkit called 'NLP Preprocessing library for Croatian texts' made by 
TakeLab (Text Analysis and Knowledge Engineering Lab) 
of Faculty of Electrical Engineering and Computing, University of Zagreb.
This toolkit is NOT included in this repository.
Please contact TakeLab (http://takelab.fer.hr/) if you want to process Croatian texts.

TakeLab toolkit contents should be put in directory preprocessing_and_classification.


This project uses IPython notebooks with Python 2.7.10.
For view only purposes HTML versions of notebooks were made.
